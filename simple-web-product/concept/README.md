# Concept

This directory captures the **product thoughts** behind this simple web product.

It is meant to hold the evolution of the idea over time:

- inception (what problem are we solving?)
- guiding principles
- mental models
- key conceptual leaps
- what we learned and how the concept changed

Guidelines:
- Keep documents short and focused.
- Prefer plain language.
- When a concept becomes a concrete decision, link it from `design/`.
