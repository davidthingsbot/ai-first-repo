# Simple Web Product (AI-first repo pattern)

This pattern describes a standalone repository structure for building a small web product with an AI coding agent and a human operator.

## Structure

- `background/` — landscape, prior art, constraints, reference notes
- `design/` — requirements, architecture, decisions
- `app/` — the actual code (frontend/backend), tests, configs
- `journal/` — month-by-month progress log and decisions-in-practice
